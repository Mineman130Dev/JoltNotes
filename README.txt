JoltNotes Lite

This is a lightweight note-taking application for macOS. This is designed to be super lightweight 
and simple to use, so whenever you need to write a idea down you could with ease!

Features:
Super simple
Small size

How to use:
As of now, there are two buttons. One for saving and one for loading, use either or for actions related to what you want to do.

*NOTE*
This is no auto save feature, so please don't forget to save your note before exiting the application.
